Detailed Description

Класс для обработки запросов на удержание (обновление) сессии.

Отвечает за взаимодействие с Redis для обновления времени жизни токенов сессий.

nlohmann::json SessionHold::HandleRequest ( const nlohmann::json & request_data)

Обрабатывает запрос на удержание (обновление) токена сессии.

Parameters
request_dataВходящие данные запроса в формате JSON, содержащие поле "token".
Returns
JSON-объект с результатом операции.

Извлекает токен из данных запроса, вызывает hold_token для обновления времени жизни токена в Redis. Возвращает успешный ответ, если токен существует, или ошибку, если токен не найден, истек или формат JSON неверен.
