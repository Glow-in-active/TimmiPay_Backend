\doxysubsection{Detailed Description}
Тестовый класс для функций запуска сервера и обработки ошибок. 

Этот класс содержит общие настройки и вспомогательные методы для тестирования различных сценариев, связанных с запуском HTTP-\/сервера и его обработкой исключений.
